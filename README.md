## Goal
- 실제 화면을 구성하기 위한 Spring MVC 설정과 Thymeleaf 기술 활용

  - Thymeleaf를 이용한 화면 출력과 반복, 제어 처리
  - 기본 객체를 이용해서 날짜, 시간 처리하기
  - 레이아웃 기능을 활용하여 템플릿 구성하기
